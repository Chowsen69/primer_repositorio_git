Comandos aprendidos durante el desarrollo

pwd

Nos dice el directorio actual en el que nos encontramos.

cd "file"

Permite introducirnos dentro de una carpeta.

git init

Le decimos que vamos a utilizar GIT en el directorio actual.

git status

Muestra el estado en el que se encuentra GIT.

git add "file"

Sirve para poder añadir un archivo para posteriormente poder hacer un commit. Otra alternativa es utilizar "git add ." para añadir todos los archivos directamente, y evitar tener que añadir uno por uno.

git commit

Nos permite hacer un commit. Una alternativa es utilizar "git commit -m 'Example message'" para añadir un comentario al commit.

git log

Nos muestra todos los commit que fuimos haciendo con el tiempo.

git branch

Muestra las versiones que tenemos del proyecto (por defecto master viene como versión principal).

git checkout "branch_name"

Permite movernos de versión en versión.

git remote add origin https://github.com/user/repository.git

Le decimos a GIT que el proyecto se va a alojar/guardar en un repositorio determinado.

git push -u origin "branch"

Lo que hace es guardar los commit en el repositorio que anteriormente le dijimos que íbamos a utilizar.
